# Cars Dealership Capstone Project

## Repository Name

cars-dealership-capstone

## Project Name

Cars Dealership

## Description

This project is a full-stack web application for a car dealership company.

The application allows users to:
- View dealership branches
- Search dealers by state
- Register and login users
- View dealer reviews
- Submit reviews
- Analyze review sentiment

## Technologies Used

Frontend:
- React
- JavaScript
- HTML/CSS

Backend:
- Django
- SQLite

Other Tools:
- Docker
- Kubernetes
- GitHub Actions
- IBM Cloud Code Engine
